What is on this page?

This page contains the resources you need to get started with coding in R. While this guide is intended for science students, the resources here are still quite general and can be used for a variety of applications. 

Each resource is hyperlinked with a description below the link on how it is intended to be used

Glossary

When learning a new technique, there is sometimes some confusion when you come across terminology you haven't hard before, even when using resources designed for beginners. Some common terms that can cause confusion among beginner R users can be found below. 

R: R is the software environment that is normally used for statistics and data visualization. Note that you cannot "interact" with R on its own. You will need to download R Studio. 

R Studio: this is the most popular application for coding in R. 

Posit: The company that maintains R recently changed their name. You may see official resources listed under R or under Posit, it is the same organization. 

Base R: This is how people commonly refer to the set of functions that come pre-installed when you download R for the first time. 

Package: These are the extra elements of code that you need to download on your own to do specific analyses. Base R is not extensive, and you will often need extra packages to complete your work. 

CRAN: Comprehensive R Archive Network. You can think of this as the app store for the different R packages and software.

These books are not for any specific functions or tasks in R. These are just general reference books. Resources for specific functions and tasks can be found under "key packages".

  • R for Data Science by Hadley Wickham; Garrett Grolemund
    ISBN: 9781491910368
    Publication Date: 2016-12-12

R for Data Science is the foundational textbook for all things R. It is available as an ebook via the McGill Library. This is not something you are meant to read cover to cover.  It contains information on basic functions and explains the different datatypes that are used within the R environment. It also offers an introduction to models and functions. This book is available through the McGill Library and via the hyperlink included above. 

  • Advanced R by Hadley Wickham
    ISBN: 1498759807
    Publication Date: 2015-09-15

Even as a beginner, there are sometimes very specific advanced functions that you require for your analysis. If this is the case, this is a great resource to get more information. You can consider this the sequel to the R for Data Science book listed above. This book is also available through the McGill Library and online through the hyperlink included above.

Important Packages

  • Packages
  • APA Citations

Packages are extra functions that you can download to do fun things in R that don't come pre-downloaded. There are many, many packages available for the various needs of R users. The packages included below are the ones that tend to be most useful for completing bioinformatic analyses in R.

ggplot2 

ggplot2  is used for data visualization. Bar charts, scatter plots, violin plots-you name it there's a way to do it with GGplot. 

  • Ggplot2 by Hadley Wickham
    ISBN: 9783319242774
    Publication Date: 2016-06-08

This is an online book that is generally considered the gold standard of GGplot references. It is written by Hadley Wickham, who is a long time R developer and chief scientist at posit. Browse the table of contents located on the left side of the page, or try using the search function in the upper left corner to find a specific resource. This resource has a lot of information that can be useful to beginners, especially if you are interested in the underlying theory and grammar of R. 

Online ggplot2 tutorial

This is very similar to the resource above, but it provides more practical examples that can be replicated on your own computer. 

ggplot2 cheatsheet

Posit creates its own cheat sheets for commonly used packages. This is the one they have generated for ggplot2. It breaks down the use of ggplot2 according to the variable types you are trying to visualize (for example, it gives you all the options you have when you have two discrete variables). This is not an extensive resource, but it couples well with the more descriptive resources linked above. 

CRAN-Documentation for ggplot2

This is the official documentation for the package. You can consider this a dictionary of all the functions that are found in the package. To beginner users, these documents can be a little intimidating, but its is good to familiarize yourself with these as they are very common and give a full overview of what is available in the package instead of just what tutorial writers have decided to highlight. 
 

Tidyverse

Tidyverse can be considered a package of packages. Tidyverse contains all the different packages you might need to clean data. There are a lot of different tasks that fall under cleaning data but the big ones are normally separating or combining columns, summarizing data and joining data in two or more separate tables. The most commonly used packages are Tidyr and Dplyr.  

Tidyr Cheatsheet

Tidyr has all the functions you need to split cells, combine cells, handle missing values and pivot tables between long and wide formats. This is the cheat sheet provided by posit that shows you all the options you have available with textual and visual explanations of what will happen with your data. 

Dplyr Cheatsheet

Dplyr has the functions you need to do different summarizing activities-for example, figuring out how many times a name appears in a column. Like the cheet sheet for tidyverse, it shows you all the options you have available with textural and visual explanations of what will happen with your data.   
 

Bioconductor

Almost every bioinformatic analysis will use the bioconductor package. Instead of doing things like cleaning or visualizing data, biocinductor will give you the resources you need to do things like microbiome or phylogenetic analyses.
